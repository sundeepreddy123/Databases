what is kafka?
  kafka is a distributed event streaming platform used to send, receive, store and process large amounts of data in real time.

Think of it like this:

KafKa  =  A high-speed messaging system for sending data between services.

it is used for:
  real-time data pipelines
  microservice communication
  logs & metrics streaming
  event-driven systems
  payment events
  gaming events
  order systems

why kafka is used?
  kafka is used when system needs:
  High throughput (millions of messages/sec)
  Real-time data
  Loose coupling between services
  Fault-tolerance
  scalable communnication
  Replayable messages
